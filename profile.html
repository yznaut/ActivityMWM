ABOUT MWM

Purpose: MWM helps movie enthusiasts organize their watchlist, track what they're currently watching, and record their thoughts on finished movies with ratings and comments.

Key Features

  • Add movies with title, genre, and release date
  • Move movies from "Currently Watching" to "Watched"
  • Rate watched movies with 1-5 stars
  • Add personal comments/reviews
  • Delete any movie entry
  • Responsive and clean modern UI
  • Persistent data using localStorage

Technologies Used

  • HTML5
  • CSS3 (Flexbox & Grid)
  • Vanilla JavaScript
  • Font Awesome Icons
  • Netlify for Deployment
